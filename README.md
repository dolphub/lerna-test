# workspace-lerna-example

## Lerna SemVer

| Commit   | SemVer |
| -------- | :----: |
| feat     | minor  |
| fix      | patch  |
| docs     |  none  |
| style    |  none  |
| refactor | patch  |
| perf     | patch  |
| build    |  none  |
| ci       | patch  |
| chore    | patch  |

### Note
- Breaking changes causes Major SemVer bump.
